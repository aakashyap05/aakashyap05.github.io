Project

Valveless Pulsejet Engine

This project focuses on the design and analysis of a Lockwood-style valveless pulsejet engine concept. The goal is to combine CAD modeling, cold-flow CFD, and MATLAB-based calculations to better understand airflow behavior, resonance estimates, fuel delivery, and expected thrust trends.

Project Goals

  • Design a manufacturable valveless pulsejet engine geometry.
  • Use SolidWorks Flow Simulation to study cold-flow behavior.
  • Use MATLAB to estimate thrust, resonance, and performance trends.
  • Plan a safe test bench setup for future physical testing.

Tools Used

SolidWorks, SolidWorks Flow Simulation, MATLAB, CAD, CFD, technical documentation.

Work Completed

  • Created and refined CAD geometry for the engine body, intake, chamber, and tailpipe.
  • Set up cold-flow CFD boundary conditions and mesh controls.
  • Developed MATLAB calculations for early performance estimates.
  • Compared design choices such as pipe diameter, intake layout, and chamber volume.

What I Learned

This project helped me connect mechanical design, fluid behavior, simulation setup, and engineering judgment in a project where geometry and assumptions heavily affect predicted performance.
