Overview

At NextGen Federal Systems, I support simulation-focused digital engineering work that sits at the intersection of software, engineering analysis, and repeatable technical workflows. My work involves writing Python scripts, working with C++, creating Docker-based environments, and using WSL to support Linux-based development on a Windows machine.

The experience has strengthened my ability to work across mechanical/aerospace engineering and software-driven simulation environments. Instead of only using engineering tools as a user, this work has pushed me to help build and improve the workflows that make simulation and data conversion tasks more reliable.

Workflow Automation Contribution

One of my key contributions was developing a Python workflow script to streamline VDB generation after a container has been built. The script was created to make the workflow easier to run, reduce manual setup steps, and lower the chance of user error when converting TIFF data into VDB files.

This kind of work is important because simulation workflows can become fragile when they depend on many manual steps. By improving the workflow around file conversion and container-based setup, the script helped make the process more repeatable, easier to use, and better suited for technical users working in a development environment.
