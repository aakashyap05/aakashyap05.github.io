Simulation Ansys / STK

Aerospace Modeling and Simulation

Worked with Ansys products to build understanding of aerospace simulation tools, including Ansys Fluent, Ansys Motion, and finite element analysis software.

One major simulation effort involved modeling a KC-135 aircraft in Ansys and running a Fluent-based analysis to support fuel consumption estimation. The resulting engineering output was then used as an input for an STK scenario, connecting vehicle-level simulation work to mission/scenario-level analysis.

Engineering challenge Connect aircraft modeling and CFD results to scenario-level analysis.

Automation Python / Docker / WSL

Workflow Automation and VDB Generation

Developed a Python workflow script to streamline VDB generation after a container has been built. The script was created to make the workflow easier to run, reduce manual setup steps, and lower the chance of user error when converting TIFF data into VDB files.

This work gave me practical experience with automation, container-based workflows, file conversion processes, and improving the usability of technical tools for engineering and simulation tasks.

Takeaway Good automation helps engineers trust, repeat, and improve technical processes.
